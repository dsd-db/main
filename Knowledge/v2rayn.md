#### 使用

1. 双击蓝色图标打开, **Subscriptions** - **Settings**;
![](./img/v2rayn1.png)
2. 点击 **Add** 新增订阅配置，**url** 处粘贴你的订阅链接;
![](./img/v2rayn2.png)
3. **Subscriptions** - **Updetes** 更新订阅;
![](./img/v2rayn3.png)
4. 右键屏幕右下角的图标,选择一个节点;
![](./img/v2rayn4.png)
5. 右键屏幕右下角的图标, **System proxy** - **Set system proxy** 开启全局代理;
![](./img/v2rayn5.png)
6. 开启全局代理后,右下角的图标会变成红色.
